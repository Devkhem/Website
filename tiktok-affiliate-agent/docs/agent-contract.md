# Agent Contract

เอกสารนี้กำหนดพฤติกรรมของ TikTok Affiliate AI Agent เพื่อให้ต่อยอดเป็นระบบจริงได้ง่าย ไม่ว่าจะทำด้วย Python, n8n, Make, Zapier, หรือเครื่องมือ workflow อื่น

## เป้าหมาย

สร้างคอนเทนต์ TikTok Shop Affiliate รายวันให้พร้อมผลิตและพร้อมโพสต์ โดยเน้นสินค้าที่มีโอกาสกดตะกร้า ไม่ใช้คำเคลมเกินจริง และมีจุดอนุมัติก่อนเผยแพร่

## Input หลัก

- `brand_profile`: กลุ่มเป้าหมาย โทนเสียง ข้อห้าม และเวลาลงโพสต์
- `product_catalog`: รายการสินค้า ราคา คอมมิชชัน รีวิว จุดขาย และลิงก์ตะกร้า
- `daily_goal`: เป้าหมายของวัน เช่น เพิ่มยอดกดตะกร้า เพิ่มยอดขายสินค้าแม่และเด็ก หรือทดสอบสินค้าใหม่
- `yesterday_metrics`: ผลลัพธ์จากวันก่อน เช่น วิว retention CTR กดตะกร้า ยอดขาย และคอมมิชชัน

## Output หลัก

- แผนคอนเทนต์รายวัน
- สคริปต์พูด 15-45 วินาที
- ข้อความบนจอ
- shot list สำหรับถ่ายหรือสร้างวิดีโอ
- caption และ hashtag
- posting queue
- บทเรียนเพื่อปรับแผนวันถัดไป

## Agent Roles

### Product Scout

เลือกสินค้าจากข้อมูลที่มี โดยดูจากราคา คอมมิชชัน รีวิว ความเข้าใจง่ายของปัญหา และโอกาสทำ hook ได้ไว

### Content Idea Agent

เปลี่ยนสินค้าให้เป็น angle ที่คนอยากดู เช่น ปัญหาที่เจอบ่อย การทดสอบจริง before/after หรือความคุ้มค่า

### Script Agent

เขียนสคริปต์ให้เหมาะกับ TikTok: เปิดแรงใน 3 วินาทีแรก เนื้อหาสั้น ไม่วกวน และมี call to action ที่ไม่แข็งเกินไป

### Video Agent

แตกสคริปต์เป็น shot list พร้อมข้อความบนจอและ note สำหรับตัดต่อ

### Publisher Agent

เตรียม caption, hashtag, เวลาโพสต์ และ checklist ก่อนกดเผยแพร่จริง

### Analytics Agent

อ่านผลลัพธ์แล้วบอกว่าควรทำอะไรซ้ำ ควรเลิกทำอะไร และควรทดสอบอะไรต่อในวันถัดไป

## Approval Gates

ต้องให้คนอนุมัติก่อนทำสิ่งเหล่านี้:

- โพสต์จริง
- ใช้ claim ด้านสุขภาพ ความงาม การเงิน หรือผลลัพธ์เฉพาะบุคคล
- เปลี่ยนราคา โปรโมชัน หรือคำอธิบายสินค้าที่อ้างอิงจากร้าน
- ลงคอนเทนต์ที่มีภาพบุคคล โลโก้ หรือเสียงที่อาจมีลิขสิทธิ์

## Success Metrics

- คลิปถูกผลิตได้อย่างน้อย 2-3 ชิ้นต่อวัน
- มีสินค้าในตะกร้าตรงกับเนื้อหาของคลิป
- CTR ไปตะกร้าเพิ่มขึ้น
- คอมมิชชันต่อคลิปหรือยอดขายต่อวันเพิ่มขึ้น
- Agent สร้างบทเรียนและปรับ angle วันถัดไปได้
